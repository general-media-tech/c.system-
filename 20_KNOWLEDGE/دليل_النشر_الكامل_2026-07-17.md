# المرحلة 4️⃣ — التحقق

1. انتظر 1–3 دقائق بعد الرفع.
2. سجّل دخولك للقاعدة الرئيسية، ثم في SQL Editor عيّن نفسك أدمن:
   ```sql
   update gmt_users set role='admin' where username='اسم_حسابك';
   ```
3. افتح صفحة `02_POS` أو `04_ADMIN_POS` كأدمن.
4. اضغط زر **🧪 الفاحص الذاتي** (أسفل اليسار) → يجب أن يصبح **أخضر**.
5. (اختياري) اضغط **🤖 الفاحص الشامل** — يجرّب كل زر وكل ترابط تلقائياً.
